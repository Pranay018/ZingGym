# Architecture

## Overview
This project is a React + Vite frontend application organized into reusable UI components.

## High-Level Architecture
Frontend (React + Vite)
│
├── index.html → Root HTML file
├── main.jsx → App entry point
├── App.jsx → Main layout wrapper
│
└── Components  
    ├── Navbar  
    ├── HeroSection  
    ├── About  
    ├── Gallery  
    ├── Trainers  
    ├── PricingSection  
    ├── Testimonial  
    ├── ContactSection  
    └── Footer
